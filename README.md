# Mamba-related-windows-builds

Unofficial self used wheels built in windows11, python3.10.11, pytorch2.4.1 with cuda12.4. Serving publics struggle finding ways to install packages that only prebuilt in linux by officials.



Compiling process learnt from https://divertingpan.github.io/post/win-mamba/

If you'd like to follow the tutorial above, remember using x64 Native Tools Command Prompt for VS to make sure MSVC working in x64.
